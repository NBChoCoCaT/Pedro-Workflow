\begin{frame}{한·영 혼용 예시}
  \begin{itemize}
    \item \textbf{WTP}(willingness to pay)와 \textbf{WTA}(willingness to accept)의 차이는 endowment effect와 측정 오차 두 가설 모두로 설명할 수 있다.
    \item 본 프로포절은 \textbf{induced value}와 \textbf{homegrown value} 두 조건을 결합한 dual design을 사용한다.
    \item 가치 유도는 induced 조건에서 \textbf{BDM} 메커니즘으로, homegrown 조건에서 \textbf{SPA}(second-price auction)로 수행한다.
  \end{itemize}

  \vspace{0.5em}
  \key{핵심:} AI-information의 효과가 두 조건에서 다르다면, 어느 채널이 작동하는지 식별할 수 있다.
\end{frame}

\begin{frame}{TikZ \& 수식 렌더링 확인}
  \begin{itemize}
    \item 수식: 개별 추정량
      \[
        \widehat{\tau} \;=\; \frac{1}{n_1}\sum_{i:T_i=1} \Delta_i
                            - \frac{1}{n_0}\sum_{i:T_i=0} \Delta_i,
        \qquad \Delta_i = \mathrm{WTA}_i - \mathrm{WTP}_i.
      \]
    \item 컬러 강조: \good{positive}, \bad{negative}, \muted{neutral}
    \item 본 슬라이드가 깨지지 않고 렌더링된다면 환경 설정 점검 완료.
  \end{itemize}
\end{frame}
