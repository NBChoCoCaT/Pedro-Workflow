# ADR-003: AI 정보 제공의 매체 — 정적 텍스트 (스크린샷·고정 메시지)

**Status:** ACCEPTED
**Date:** 2026-05-26
**Context:** `/interview-me` Phase 3 (Q9), 03_AIWTPGap 연구 디자인 단계

## Problem

AI(LLM)가 피험자에게 가치 관련 정보를 *어떻게* 전달할 것인지 결정해야 한다. 매체 선택은 (i) 실험 통제 가능성, (ii) 외부타당도, (iii) AI 메시지 내용의 사전등록 가능성에 직접 영향을 준다.

## Options considered

### Option A: 정적 텍스트 (스크린샷 또는 고정 메시지)

연구자가 사전에 LLM에 질의하여 응답을 받아 *고정 텍스트로* 모든 피험자에게 동일하게 제시.

**Pro:**
- *완전한 stimulus control*: 모든 피험자가 동일 메시지 → arm 간 confound 없음.
- 메시지 내용 100% **사전등록 가능** — Q12-(4) "AI 메시지의 내생적 내용" 위협 직접 제거.
- 메시지 검토(연구자가 메시지의 가치 함량·앵커 강도 사전 정량화 가능).
- 비용·기술 부담 최소.

**Con:**
- 외부타당도 감소: 실제 소비자의 ChatGPT 사용은 *대화적·맞춤형*임. 정적 텍스트는 그 경험과 거리가 있음.
- 피험자가 메시지를 "AI가 *방금* 생성한 답변"으로 인식할지 의문 → demand effect의 *방향*이 달라질 수 있음.

### Option B: 실시간 LLM 호출 (피험자가 직접 질의)

피험자가 자신의 질문을 입력 → LLM이 실시간 응답 → 피험자가 응답을 읽고 valuation.

**Pro:**
- 외부타당도 강함. 실제 소비자 사용 환경과 일치.
- AI 메시지의 *개인화* 효과 측정 가능.

**Con:**
- *Stimulus control 상실*: 피험자마다 다른 질문 → 다른 응답 → arm 간 비교 불가.
- LLM 응답의 stochasticity (temperature > 0)로 인한 추가 분산.
- 사전등록 어려움 — 메시지 내용을 미리 적시할 수 없음.
- API 비용 + 인프라 부담.
- Hallucination 위험 (피험자에게 *틀린 가격 정보* 제공될 가능성, IRB 우려).

### Option C: 음성 인터페이스 (TTS / 대화형)

LLM 응답을 음성으로 전달.

**Pro:** 매체 효과 자체 연구 가능. 외부타당도 ↑ (음성 챗봇 확산).
**Con:** 통제 더 어려움. 본 연구의 1차 질문(정보 효과)이 매체 효과와 교란.

## Decision

**Chose:** Option A — 정적 텍스트 (스크린샷 또는 고정 메시지).

**Rationale:** 본 연구의 1차 식별 대상은 *정보 보정 vs 앵커링 vs source* 채널 분해이며, 이를 위해서는 **메시지 내용의 완전한 통제와 사전등록**이 필수다. Option B의 외부타당도 이점은 매력적이나 — stimulus control 상실은 본 연구의 식별 전략 자체를 무너뜨린다 (각 피험자가 사실상 다른 처치를 받는 셈). 외부타당도는 후속 연구(Option B 형식)에서 보완.

## Consequences

- **모든 AI 메시지(arm별)는 데이터 수집 *전*에 결정되고 사전등록된다.**
- 메시지 작성 절차:
  1. 연구자가 LLM(GPT-4 또는 Claude)에 질의를 던져 응답 받음
  2. 응답 텍스트를 *그대로* 또는 *최소 편집*으로 stimulus화
  3. arm 간 메시지 길이·톤·구체성을 통제하기 위한 *작성 매뉴얼* 작성
  4. pilot에서 메시지 읽기 시간·이해도 측정
- 피험자에게는 "이 메시지는 OpenAI GPT-4(또는 Anthropic Claude)에 [질문 텍스트]를 입력했을 때 받은 응답입니다"라고 명시적 source 표시 → AI 출처 인식 통제.
- **외부타당도 한계 인정 사항**으로 페이퍼 §Limitations에 명시.

## Rejected alternatives — why not

- **B (실시간 LLM 호출):** stimulus control 상실로 식별 전략 붕괴. 외부타당도는 후속 연구로.
- **C (음성):** 매체 효과와 정보 효과의 교란. 본 연구 범위 밖.
